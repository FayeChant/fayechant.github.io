Stereoframework module¶

Description¶

Author:Faye Chant <gy16fc@leeds.ac.uk>
Date:2019-09-25
Revision:1
Summary:The stereoframework module sets up the framework for the SHPrep.py module. It contains the following classes:
  • Files
  • Field
  • Features, and
  • Fishnet.

Use¶

This module is called in SHPrep.py. The user will be prompted to input the working directory, and an input points file.

Classes and Modules¶

class Stereoframework.FIELD(fieldname, fieldtype, decplaces, fieldlen, expression)[source]¶

Bases: object

The FIELD class sets up the following field attributes for new fields; fieldname, fieldtype, decplaces, fieldlen, expression.

field name = text string

field type = DOUBLE, FLOAT, SHORT, LONG, STRING

decimal places = the number of decimal places for field types of DOUBLE or FLOAT

field length = number of characters for field types of STRING

“expression” will be used in the field calculator to copy over the original field name (re-format it) or set a default value

This method uses the field calculator to migrate the existing fields to the new correctly formatted fields

class Stereoframework.Files[source]¶

Bases: object

The “files” class sets the file paths of the files to use.

It will generate the following files:

Original point file - input by user (Default “OBSTRUCTIONS.shp”)

Final - reformatted with unwanted fields removed, sorted by geographic location - input by user (default = OUTOBSTRUCTIONS_final.SHP”)

Fishnet - polygon file to define cells for splitting up the point file.

Fishnet_Count - Spatial join of Final and Fishnet with count of points in each cells

Copy - temporary file. A copy of Original

Tempfishnet - temporary file. Used for optimising fishnet

Fishnet2Obstr - temporary file. Used to intersect points with Polygon and assign cell number for exporting shapefiles of points within individual cells.
